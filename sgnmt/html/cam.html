Module contents¶

This is the main SGNMT package. It contains the packages

  • blocks: Code with dependency on the blocks framework
  • decoding: Interfaces and implementations of decoding strategies
  • predictors: Scoring modules

Direct modules are:

  • io: Contains I/O related code such as output handlers
  • utils: Common helper functions and global constants
