Detailed Description

Contains data for statistics, i.e. allows timing actions and methods called during execution.

Remarks
Timing in ms. Each measurement is assigned a specific key. The key is expected to be meaningful to the user. Finally, methods provided to dump a list of pairs: key1:time1 key2:time2 ...
